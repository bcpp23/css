Trabalhando com HTML e CSS

Algumas das principais tecnologias e linguagens Web são o HTML, CSS, Javascript e PHP. São linguagens usadas para estruturar, estilizar e implementar comportamentos dinâmicos aos Web Sites. Também é muito comum o uso de sistemas gerenciadores de banco de dados, como o MySQL, na implementação de sites.

Abaixo você encontra uma lista com alguns programas utilizados para o desenvolvimento de sites:

  • Brackets
  • Notepad++
  • Sublime
  • Dreamweaver
